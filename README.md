# Next.js 16 Starter Template

A lightweight, TypeScript-based Next.js 16 template with commonly used libraries preconfigured for quick project setup.

---

## Features

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS + ShadCN UI
- Zustand (state management)
- React Query + Axios (data fetching)
- NextAuth.js (authentication)
- React Hook Form + Zod (forms & validation)
- Framer Motion, React Hot Toast, Lucide Icons
- Next Themes (dark/light mode)
- date-fns (utilities)

---

## Quick Start

```bash
pnpm create next-app -e https://github.com/ArhamMurtaza/my-next-template my-app
cd my-app
pnpm install
pnpm dev
```
